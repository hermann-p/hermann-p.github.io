Universeller String

  • k-universell \(\leftrightarrow\) jedes mögliche k-mer genau einmal
  • Beispiel: 0001110100
    • 3-universell \(\to\) enthält
    • 000, 001, 011, 111, 110, 101, 010, 100

Universeller String

00011101 zirkulär, 3-universell

de Bruijn Graph

  • deBruijnk( <seq> ): Graph aller k-mere von seq
  • Knoten: (k-1)-mere (Präfix, Suffix)
  • Kanten: k-mere
  • Identische Knoten identifizieren, zusammenführen

Beispiel

deBruijn3( 0001110100 )

Sequencerfragmente

Vorteile

  • Jede Kante 1x statt jeden Knoten
  • Euler- statt Hamiltonpfad
  • existiert wenn |in| = |out|
  • linearer Zeitaufwand
  • "almost balanced" \(\to\) Kante hinzufügen \(\to\) balanced
